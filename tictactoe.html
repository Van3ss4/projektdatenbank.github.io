3D 4x4 Tic-Tac-Toe

  • Februar 2020

Projektbeschreibung

Der LED-Cube wurde aus P9823 LEDs (baugleich WS2812b) zusammengebaut, die - richtig verdrahtet - wie ein LED-Streifen mithilfe eines Arduinos und der Neopixel-Library angesteuert werden können.

Mit dem Controller navigiert man durch das Menü und kann zwischen dem Ein- und Zweispielermodus auch ein Untermenü mit Einstellungen (Spielerfarbe und LED-Helligkeit) auswählen.

Um eine Position zu setzen, verschiebt man einen blinkenden Marker mit den Buttons auf dem Controller und bestätigt die Position schließlich. Dann kann entweder der andere Spieler seine Position setzen oder das Programm generiert einen Move.

Prüfalgorithmus: "Hat jemand gewonnen?"

Es wird immer nur geprüft, ob der Spieler, der zuletzt einen Move gesetzt hat, gewonnen hat. Das Programm durchläuft das Spielfeld, das als dreidimensionales Array abgespeichert ist. Befindet sich ein Spieler an einer Position befindet sich doch eine "1" oder "2", sonst eine 0. Findet das Programm den Spieler, durchsucht es die Nachbarfelder. Wird ein Nachbar gefunden, wird dessen relative Position zum ursprünglichen Punkt gespeichert und auf die absolute Position des Nachbars addiert. Wird ein weiterer passender Nachbar gefunden, wird die Reihe auf diese Weise immer weiter verlängert, bis sie lang genug (voll) ist. Dann hat dieser Spieler gewonnen.

Einspielermodus: Computer-Move

Das Programm prüft zuerst, ob es mit einem Move gewinnen kann. Ist das der Fall, hat dieser Move die höchste Priorität und wird gesetzt. Wenn nicht, prüft es, ob sein Gegner mit einem Move gewinnen kann, und verhindert das gegebenenfalls, indem es die Position mit seinem Punkt blockiert. Kann weder das Programm, noch der Spieler mit einem Move gewinnen, wird geprüft, ob das Programm in zwei Moves gewinnen könnte. Es "plant" also voraus. Ist das auch nicht möglich, wird eine zufällige freie Position gesetzt.

Controller

Er wurde aus 7 Drucktastern zusammengebaut. Je nachdem, in welchem Untermenü man sich befindet, haben die Tasten verschiedene Funktionen.

Innenleben

Das TFT-Display und die Controller-Knöpfe werden von einem Arduino Mega angesteuert, der über ein Relais die Stromversorgung für den LED-Cube regelt
